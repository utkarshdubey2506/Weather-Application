# Weather App - React.js

A responsive weather application built with React.js that provides real-time weather information with enhanced features.

## Features

- **Current Location Weather**: Automatic weather detection based on user's location
- **City Search**: Live search with autocomplete suggestions for any city worldwide
- **Recent Searches**: Quick access to last 2 searched cities
- **Bookmark Cities**: Save up to 3 favorite cities with one designated as "home"
- **Weather Details**: Temperature, humidity, visibility, and wind speed
- **Animated Weather Icons**: Dynamic weather animations based on current conditions
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Weather Data Displayed

- **Temperature**: Current temperature in Celsius
- **Weather Condition**: Clear, cloudy, rainy, snowy, etc.
- **Humidity**: Air moisture percentage
- **Visibility**: Atmospheric visibility in kilometers
- **Wind Speed**: Current wind speed in Km/h
- **Location**: City name and country
- **Real-time Updates**: Weather data refreshes automatically

## Setup

```
npm i && npm start
```
